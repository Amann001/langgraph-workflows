# LangGraph Workflows

A collection of workflow implementations built using LangGraph, LangChain, and Groq.

## Implementations

### Sequential Workflows

* BMI Calculation Workflow
* Simple LLM Workflow
* Prompt Chaining Workflow

### Parallel Workflows

* Parallel Batsman Analysis
* Parallel UPSC Essay Evaluation

### Conditional Workflows
- Quadratic Equation Workflow
- Customer Review Reply Workflow

## Tech Stack

* Python
* LangGraph
* LangChain
* Groq
* Pydantic
* Jupyter Notebook

## Concepts Covered

- State Management
- Sequential Workflows
- Parallel Workflows
- Conditional Workflows
- Conditional Routing
- Structured Outputs
- Workflow Orchestration
- LangGraph StateGraph
- Groq Integration
